# Completed Tasks

## Task 1 – Rule-Based Chatbot

A simple AI chatbot that responds to user queries using predefined rules and conditional statements.

**Features**

* Responds to greetings and basic questions
* Provides current date and time
* Handles simple conversational inputs

**Technologies Used**

* Python
* Rule-Based NLP

---

## Task 2 – Tic-Tac-Toe AI

An AI agent that plays Tic-Tac-Toe against a human player using the **Minimax Algorithm**.

**Features**

* Human vs AI gameplay
* Unbeatable AI logic using Minimax
* Command-line interface

**Technologies Used**

* Python
* Game Theory
* Minimax Algorithm

---

## Task 4 – Recommendation System

A simple recommendation system that suggests movies based on their similarity using content-based filtering.

**Features**

* Movie recommendation based on genre similarity
* Uses cosine similarity for comparison

**Technologies Used**

* Python
* Pandas
* Scikit-learn

---

## Task 5 – Face Detection and Recognition

A computer vision application that detects human faces in real-time using a webcam.

**Features**

* Real-time face detection
* Draws bounding boxes around detected faces

**Technologies Used**

* Python
* OpenCV
* Haar Cascade Classifier
